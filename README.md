# To-Do List App

A simple, clean, and responsive to-do list application built with vanilla HTML, CSS, and JavaScript.

## Features

- Clean and modern user interface
- Responsive design that works on desktop and mobile
- Ready for task management functionality
- Accessible and user-friendly

## Project Structure

```
├── index.html      # Main HTML structure
├── styles.css      # CSS styling and responsive design
├── app.js          # JavaScript functionality
└── README.md       # Project documentation
```

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Start adding your tasks!

## Usage

- **Adding Tasks**: Type your task in the input field and click "Add" or press Enter
- **Responsive Design**: The app works seamlessly on both desktop and mobile devices

## Development

This project uses vanilla web technologies:

- **HTML5**: Semantic structure with accessibility in mind
- **CSS3**: Modern styling with Flexbox and CSS Grid
- **JavaScript (ES6+)**: Clean, modular code structure

### File Descriptions

- `index.html`: Contains the basic structure with header, input area, and list area
- `styles.css`: Responsive styling with a clean, modern design
- `app.js`: JavaScript foundation with event handling and DOM manipulation setup
- `README.md`: Project documentation and setup instructions

## Browser Compatibility

This app works in all modern browsers:
- Chrome 70+
- Firefox 65+
- Safari 12+
- Edge 79+
